## Features

- TypeScript & TSLint

## Geting started

```bash
git clone https://github.com/MichaelOlatunji/blog-gatsby.git
```

# Install
```bash
npm install
or
yarn install
```

## Development

```bash
# develop 
npm start
or
yarn start

# build
npm run build
or 
yarn build
```
